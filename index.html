FloodlightControllerExperiments

MACAddress Tracker of hosts using Floodlight Controller

View on GitHub Download .zip Download .tar.gz

This is a quick sample of floodlight application.

Some forks have some troubles in running this basic sample. The key to these problems is that your floodlight version does not match the Official Guide of floodlight. For example, you are using floodlight V1.1 but your Official Guide is V0.9. Since, many changes are applied in the new version. Following the old Guide could not works any more.
